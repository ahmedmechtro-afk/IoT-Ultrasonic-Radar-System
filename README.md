# IoT-Ultrasonic-Radar-System
ESP32-based IoT Ultrasonic Radar System using HC-SR04, SG90 servo, Processing visualization, and ThingSpeak.
# IoT Ultrasonic Radar System

This project is an ESP32-based IoT Ultrasonic Radar System that detects objects using an HC-SR04 ultrasonic sensor mounted on an SG90 servo motor. The system scans the surroundings, displays objects on a digital radar screen, and uploads distance data to ThingSpeak for real-time visualization.

## Features
- ESP32 microcontroller based system
- Ultrasonic distance measurement
- Servo-controlled angular scanning
- Processing-based digital radar display
- ThingSpeak IoT cloud integration

## Hardware Components
- ESP32 (ESP32-S3)
- HC-SR04 Ultrasonic Sensor
- SG90 Servo Motor
- Breadboard and Jumper Wires

## Software Tools
- Arduino IDE
- Processing
- ThingSpeak IoT Platform
